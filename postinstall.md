# SNAPSPUNK Post-Installation Guide

**DO THIS IMMEDIATELY AFTER INSTALLING** — your list **will crash constantly** or have broken UI/cosmetics if you skip any step.

## 1. Antivirus & Overlays (Cyberpunk 2077 ONLY)
- Add your entire install folder to Windows Defender exclusions (or fully disable 3rd-party antivirus).
- Disable **ALL** overlays: Steam, Discord, NVIDIA/AMD, Xbox Game Bar, etc.

## 2. Launch with Mod Organizer 2
- Never launch the game directly from Steam.
- Always use the **Run** button in MO2 with the correct profile selected:
  - SNAPSPUNK-2077 EBB → use EBB profile
  - SNAPSPUNK-2077 Solo → use Solo Body profile
  - SNAPSPUNK-FO4 Base / SS → use the matching profile

## 3. Cyberpunk 2077 – Required First-Run Steps
- Run the game once through MO2 so CET creates the `plugins` folder.
- Open the **Redscript / CET Mod Settings** menu (default key: configurable in CET) and apply any listed defaults.
- Enable all Atelier shops in the Virtual Atelier menu.

## 4. Fallout 4 – Required First-Run Steps (Magnum Opus Style)
- Run the game once through MO2 so the MCM and all UI mods initialize.
- Open the in-game **Magnum Opus Beginner's Guide terminal** (or holotape) outside Vault 111.
- Apply BiRaitBec Texture Optimization / WorkBase Improved if present in your list.

## 5. Body-Type Specific (Cyberpunk Only)
- EBB Profile: Make sure **only** EBB-related clothing, presets, and Atelier shops are enabled.
- Solo Body Profile: Make sure **only** Solo Body-related clothing, presets, and Atelier shops are enabled.
- Never mix body replacers between profiles.

## 6. Final Checks
- Verify you are on the **latest game version** (Wabbajack handles downgrades automatically for FO4).
- Check the in-game MCM / Redscript menus for any “Enable” toggles.
- Create a new save — do **not** load an old save from another list.

**If the game still crashes after these steps**, join the Discord and post your MO2 log + profile name. Read the FAQ first.
