### General / Installation

**Q: What version of the game do these lists use?**  
**A:**  
- **Cyberpunk 2077** → Latest Steam version + Phantom Liberty DLC (no downgrade needed).  
- **Fallout 4** → 1.10.163 (pre-Next-Gen). The downgrade is handled automatically by Wabbajack.

**Q: Where is the full load order / mod list?**  
**A:** Load Order Library links will be added to the README after initial testing. For now, open MO2 and check the left pane — the numbered separators make everything easy to find.

**Q: My game crashes on launch from MO2. Help!**  
**A:**  
1. Disable **ALL** overlays (Steam, Discord, NVIDIA/AMD, Xbox Game Bar).  
2. Add the entire SNAPSPUNK install folder to Windows Defender exclusions (or temporarily disable third-party antivirus).  
3. Run the game once through MO2 so CET/REDmod/MCM initialize.

**Q: Do I need a Nexus Premium account?**  
**A:** No — Wabbajack works without it, but downloads will be slower.

### Cyberpunk 2077 Profiles (EBB vs Solo Body)

**Q: Why are there two separate profiles for Cyberpunk?**  
**A:** To keep body replacers, presets, clothing, and Atelier shops completely conflict-free. Switch profiles in MO2 when you want to change bodies.

**Q: Can I mix EBB and Solo Body mods?**  
**A:** No. Only enable body-specific clothing/presets/Atelier shops in the matching profile. Mixing will cause invisible items or crashes.

**Q: How do I switch from EBB to Solo Body (or vice versa)?**  
**A:**  
1. Save your current game.  
2. Switch MO2 profile.  
3. Start a **new game** (body replacers are not save-safe).

### Fallout 4 Profiles (Base vs SS)

**Q: Why two Fallout 4 profiles?**  
**A:** The **Base** profile is pure Magnum Opus-style (story + tactical, no settlements). The **SS** profile adds full Sim Settlements 2 integration.

**Q: Can I add Sim Settlements to the Base profile later?**  
**A:** Yes, but it is safer and cleaner to use the dedicated SS profile.

### Gameplay & Hotkeys

**Q: What are the important hotkeys?**  
**A:**  
- **Cyberpunk** — Check CET Mod Settings + individual mod pages (Quickhack hotkeys, Nitrous, Flashlight, etc.).  
- **Fallout 4** — V = 1st/3rd person, T = Wait, N = Upgrade SS2 plots (SS profile), Insert = Workshop extra selection.

**Q: Recommended difficulty?**  
**A:** **Very Hard** for both games. The lists are balanced around it.

### Crashes & Performance

**Q: Game still crashes after following post-install guide.**  
**A:** Post your MO2 log + exact profile name in Discord. Include whether you are on EBB/Solo or Base/SS.

**Q: FPS issues or stuttering?**  
**A:** Installation must be on SSD/NVMe. Cyberpunk installation on HDD will cause heavy stuttering.

### Appearance & Body Replacers

**Q: My character looks wrong / black face / missing textures.**  
**A:**  
- Make sure the correct body replacer is enabled **and** all matching presets/clothing are enabled for that profile.  
- Run the game once after enabling new appearance mods.

**Q: Can I add my own character presets?**  
**A:** Yes — place them in the `12 - Character Presets` separator. They will survive updates if you name the mod `[NoDelete]`.

### Virtual Atelier Shops

**Q: Where do I find the new clothing/weapons?**  
**A:** Any computer → **Net** → **Virtual Atelier** icon. Shops are grouped by modder (nc fashion, VEEGEE SHOP, etc.) in the separators.

**Q: Some shops are body-type specific.**  
**A:** Yes — notes are in the separator names and in the post-install guide. Only enable the matching shops for your current body profile.

### Sim Settlements (FO4 SS Profile Only)

**Q: How do I start building settlements?**  
**A:** Follow the in-game Sim Settlements 2 tutorial quest. All chapters and plots are enabled in the SS profile.

**Q: I broke Workshop Plus and can’t fly.**  
**A:** Re-enable Workshop Plus in the 30 - Settlements section or check the MCM for reset options.

### Updating & Mod Changes

**Q: How do I update the list?**  
**A:** Download the new .wabbajack file and install over the top (check the Overwrite box). Wabbajack will delete anything not part of the list.

**Q: Can I add my own mods?**  
**A:** Yes — put them in the correct separator and rename the mod folder `[NoDelete]` so they survive updates.

**Q: Can I remove or update individual mods?**  
**A:** You can, but you are on your own for compatibility. Rule-11 style support only.
