# SNAPSPUNK Changelog

All updates follow **Semantic Versioning** (exactly like Magnum Opus):

- **x.0.0** (major) → Requires a new game.
- **x.x.0** (minor) → Usually requires a new game.
- **x.x.x** (patch) → Can be applied to existing saves (backup first!).

## 2026/03/27 - Version 1.0.0 (Initial Release)

**NOTES**
- First public release of SNAPSPUNK modlists.
- Four profiles released simultaneously:
  - SNAPSPUNK-2077 EBB
  - SNAPSPUNK-2077 Solo Body
  - SNAPSPUNK-FO4 Base
  - SNAPSPUNK-FO4 SS
- All lists are story-driven and tactical, inspired by Chrome & Blood (Cyberpunk) and Magnum Opus (Fallout 4).
- Wabbajack installation fully supported.
- MO2 separators, post-install guide, and GitHub repo set up for easy community use.
- Open permissions — fork and modify freely (just rename and credit).

**Known limitations at launch**
- Full load-order library links and detailed mod breakdown will be added after first community testing round.
- Banners and images are live in the repo.
